投资逻辑框架

样机版 | 2026-07-09 | 逻辑与数据一一对应
1. 存储需求预测
  • HBM: AI GPU 拉动，按客户、代际、容量拆分。
  • DRAM: 服务器优先，PC/手机作为弹性项。
  • NAND: 复苏慢于 DRAM，作为利润修复补充。
2. 存储供给预测
  • 2025-2030 按三星/海力士/美光 wafer 与 byte 表。
  • 跟踪 HBM 转产挤出普通 DRAM 的供给约束。
  • Capex、良率、先进封装为关键约束。
3. 内存墙构成
  • 列过去十年算力速度与存储带宽速度之比。
  • 判断未来算力继续拉开带宽，HBM 价值量上移。
  • 用 GPU 代际、带宽/容量需求做验证。
4. 竞争格局
  • HBM 市占率、认证进度、客户份额逐季跟踪。
  • 判断海力士领先窗口期长度及被追赶风险。
  • 三星、美光 HBM3E/HBM4 放量是关键变量。
5. LTA 进展与占比
  • 列订单事件、锁量、价格机制、交付时间节点。
  • 跟踪 LTA 占 HBM/DRAM 收入比例。
  • 判断利润可见度和估值溢价持续性。
6. 盈利计算公式
  • 收入 = bit 出货 x 每 Gb 价格 x mix。
  • 毛利 = 收入 x 毛利率；净利扣除费用/税率。
  • 目标市值 = 净利 x PE；再换算目标股价。
